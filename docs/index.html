COVID-19 Outbreak Simulator

Evaluate the effectiveness of preventative and interventive measures against the spread of SARS-CoV-2 virus.

Learn more

A population-based outbreak simulator

COVID-19 Outbreak Simulator is a population-based simulator that simulates the spread of SARS-CoV-2 virus in a dynamic population, subject to preventative measures and post-outbreak action plans such as quarantine, testing, and removal of symptomatic cases.

Powerful Simulator

A simulator with a powerful command line interface that simulates COVID-19 outbreaks in a heterogeneous population, with dynamic modeling parameters and multiple preventative and interventive actions.

Application-oriented Notebooks

Applications in executable Jupyter Notebook format that include narratives, simulation steps, and analysis of outputs. providing a higher-level interface for particular simulation scenarios.

Extensible by Plugins

The simulator allows the use of plugins to define customized actions, policy, and flow of individuals, to create complex simulation scenarios for your particular applications.
